Project Based Learning II

Dr. Nikhil Mhala

Assignment 1

  • Write a program for Iterative and Recursive Binary Search and verify the time complexity.
  • Write a program to perform quick/merge sort using Divide and Conquer strategy and verify the time complexity.

Assignment 2

  • Consider a friend's network on Facebook social web site. Model it as a graph to represent each node as a user and a link to represent the friend relationship between them using adjacency list representation and perform DFS and BFS traversal on the above graph.

Assignment 3

  • Write a program for Minimum Spanning Trees using Greedy approach (Kruskal's / Prim's algorithm).

Assignment 4

  • Write a program to implement A VL tree

Assignment 5

  • Implement direct access file using hashing (linear probing with and without replacement) perform following operations on it a) Create Database b) Display Database c) Add a record d) Search a record e) Modify a record

Assignment 6

  • Implement 0/1 Knapsack OR Traveling Salesperson problem using Dynamic Programming.

Assignment 7

  • Write a program for Hamiltonian Problem OR N-Queen problem using Backtracking approach

Assignment 8

  • Write a program for Huffman Coding OR 0/1 knapsack problem using Greedy approach.
