Internet of Things Lab

Assignment 1

    Study of loT architecture, development platforms and various ARM SOCs such as Raspberry Pi - Pisco 6yuy/ ESP8266 boards / Arduino Uno etc. To perform OS installations used to bind IOT devices

Assignment 2

    To Interface any sensors such as Temperature or Ultrasonic or Gas sensors with Raspberry Pi - Pisco 6yuy/ ESP8266 boards / Arduino Uno etc, and demonstrate data acquisition

Assignment 3

    To interface any actuators such as DC/ Servo / Stepper motor, relays etc. with Raspberry Pi - Pico/ ESP8266 boards / Arduino Uno.

Assignment 4

    To demonstrate MQTT/COAP protocols using message broker to subscribe and publish sensor data using Raspberry Pi-Pico/ ESP8266 boards / Arduino Uno.

Assignment 5

    To sense the data from sensors, send, store data in cloud system and further demonstrate basic presentation and analysis on the data. Also demonstrate actuator interfacing and control with cloud loT. Client side, you may use

Assignment 6

    Raspberry Pi-Pico/ ESP8266 boards / Beagle board/ Arduino Uno. You may use Thing Speak, Ubidots or any suitable cloud system

Assignment 7

    To Demonstrate IoT web application, to acquire and disseminate sensor data using, Web Server in IoT platform (loT device as Web Server) and analyse benefits of such a setup

Assignment 8

    Write or use mobile phone application to monitor and control an loT appliance

Assignment 9 : loT Mini Project: Real life loT application design & Implementation

    Students are required to design and build a mini project for any suitable loT application domains such as in the Healthcare, Manufacturing, Agriculture and Food, Insurance, Governance, Forest conservation, Transportation and Vehicles, Process management, Real estate and land records, Energy, Retails, Logistics, Education etc. Students group should submit the demonstration and prepare the 15-20 page plagiarism free report in pdf format.
