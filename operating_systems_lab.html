Operating Systems Lab

Assignment 1: Linux Commands and Shell Scripting

    Write a program to implement Shell Scripting - Arithmetic operations

Assignment 2: Process Management (Process control)

  • Write a program using fork to create a child process. The parent process should sort elements in ascending order and child process should sort elements in descending order.
  • Orphan Process
  • Zombie process

Assignment 3: Process Management (Scheduling)

    Write a program to simulate First Come First Serve (FCFS) (Non Pre-emptive) or Shortest Remaining Time First (SRTF) (Pre-emptive) CPU Scheduling algorithms

Assignment 4: Process Management (Inter-process Communication)

    Implement Pipe and / Shared Memory Concept

Assignment 5: Process Synchronization (Deadlocks)

    Write a program to simulate Bankers algorithm for deadlock avoidance.

Assignment 6: Process Synchronization (Semaphore)

    Implement Readers- Writers or Producer-Consumer Problem.

Assignment 7: Memory Management (Page Replacement Algorithms)

    Write a program to simulate FIFO and Least Recently Used (LRU) page replacement algorithm.
